- grep 命令
    grep moon monkey 
        在 monkey 文件夹找 moon

    grep -v moon monkey
           -v 逆向操作
    grep -c moon monkey       
           -c 符合条件的行数
    grep -i my monkey
           -i 大小写忽略

    grep -n monkey monkey
            -n 行号输出  在monkey文件下查找monkey
    grep '^J' monkey
            ^    以 J 为开头的
    grep '呢$' monkey
            $  以 呢 结尾的
